Projects Overview

ERP Student-Teacher Organizational Platform

A web-based internal ERP system designed to streamline academic and administrative activities between students, teachers, and administrators.

Key Features:

  • Developed responsive UI components using React.js for modules like student registration, student profile.
  • Collaborated with backend team and integrated REST APIs for data flow between frontend and backend (e.g., fetching student details)
  • Created forms with validations using React Hook Form and styled them with Ant Design, a react UI design Framework.
